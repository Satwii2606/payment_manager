# 💳 **Payment Processor**

## 📌 **Overview**
This project is a **Spring Boot backend application** that simulates basic payment operations. 
It provides REST APIs to handle payment methods like **UPI** and **Net Banking**. 
The application runs on a local server and can be tested using a browser or tools like Postman.

## 🚀 **Features**
- **REST API for payment processing**
- Supports **UPI** and **Net Banking**
- **Lightweight and easy to run**
- Clean backend structure (**Controller, Service, Model**)

## 🛠️ **Tech Stack**
- **Java**
- **Spring Boot**
- **Maven**
- **REST APIs**
- **Embedded Tomcat**
